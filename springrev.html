Multi-Mode Filters: the high-pass problem

April 7, 2025

\( H(s) = \frac{1}{s+1} \)

1 — Introduction

Short summary: This document explores the technique of pole-mixing (multi-mode filters built by summing weighted outputs of cascaded single-pole low-pass stages) and shows practical implementations using SSI2140/SSI2164 filter cells combined with the SSI2190 voltage-controlled mixer. The PDF this page is based on contains equations, Bode plots, diagrams, tables of coefficient sets and measured sweeps used throughout the design discussion.

2 — Multi-mode Filters (MMFs)

This section derives the MMF idea mathematically using Laplace transfer functions. The basic normalized single-pole responses considered are:

    1LP(s)=1/(s+1)
    1HP(s)=s/(s+1)
    1AP(s)=(s-1)/(s+1)
    1NT(s)=(s^2+1)/(s+1)^2

The document shows how sums of input and successive low-pass outputs form arbitrary numerators over (s+1)^n denominators, enabling many classical filter types (LP, HP, AP, NT, phasers, etc.) just by changing five gains A..E.

3 — Addition of resonance

The position where the resonant feedback is injected matters. The PDF demonstrates that injecting feedback after the A path leads to incorrect DC behavior for HP responses — instead the correct behavior is obtained when the feedback is summed into the input path (before A). Several diagrams and Bode plots illustrate these cases.

4 — Accuracy matters: HPF curves

Key point: DC attenuation for HPFs depends on the algebraic sum of A..E. Any mismatch in gains shows up directly as non-zero DC gain, reducing low-frequency attenuation dramatically. The PDF quantifies how a 1% mismatch in a coefficient can convert an ideal infinite attenuation (= -∞ dB) into e.g. -40 dB for a simple 1HP case, or about -24.4 dB for some 4HP cases.

5 — Accuracy matters: SSI2140

The SSI2140 emits good audio but presents variable mismatches as the cutoff frequency is swept. A test circuit subtracts the first pole output from input to make a 1HP. Trimming at a single fc cancels constant offsets but cannot completely remove frequency-dependent DC shifts introduced by the IC.

6 — Filter implementation: SSI2164 + SSI2190

To minimize variable mismatches, the author prefers SSI2164-based filter cells. For mixing the A..E coefficients the SSI2190 is used. The document describes scaling strategies and provides a practical table of presets.

7 — Trimming and measurements

  1. Set trimmers to midpoint.
  2. Select outputs A and B; adjust trimmer B to match 1HP at high fc.
  3. Select B and C; adjust C to match 1LP+1HP shape.
  4. Repeat for trimmers D and E.
  5. Trimmer F is adjusted for amplitude similar to other channels.
